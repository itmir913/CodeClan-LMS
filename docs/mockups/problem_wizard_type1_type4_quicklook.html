유형① 실행결과 맞히기와 유형④ 빈칸채우기의 3단계 설정 화면 핵심 차이점을 빠르게 보여주는 목업. ①은 정답 코드와 노출 방식 토글, ④는 Monaco 에디터의 편집구간 마커 지정 도구를 보여줌

①실행결과 맞히기 — 핵심 차이
for i in range(1, n+1):
    print(i, end=' ')
n = 5→ "1 2 3 4 5 "
n = 3→ "1 2 3 "
④빈칸채우기 — 핵심 차이 (MonacoMarkerEditor)
텍스트 드래그 선택 후
def factorial(n):
    if n <= 1: return 1
    return n * factorial(n-1)

print(factorial(5))
구간#1 (함수 본문) 구간#2 (호출부)
↓ 나머지 설정은 ②와 동일 (TestCaseEditor / LanguageSelector / AdvancedLimitsPanel / StructureCheckPanel 재사용)
